Features

Marp is the simplest presentation writer with Markdown.

  • Create presentation slides by Markdown
  • Cross-platform (Supports Windows, Mac OS X and Linux)
  • Live preview with 3 modes (Markdown / 1:1 Slide / Slides list)
  • The most simple slide theme, based on GitHub style
  • Export slides as PDF

Write the presentation with like a feeling of Markdown editor.

How to write slides?

A basically text format is based on Github Flavored Markdown.

Marp has tiny change of horizontal ruler to create multi-page presentation. You can split page of presentation by ---. This format is similar to Deckset.

# Slide 1

foobar

---

# Slide 2

foobar

Please refer the slide example.
